# Login System

A basic login system to learn more about API's

## Tecnologies
- Node.js
- Express
- SQLite3
- JavaScript
- HTML/CSS

## Functionalities 
- Registration of users (Name, Email, Password) 
- Password Validation 
- REST API for frontend/backend communication
